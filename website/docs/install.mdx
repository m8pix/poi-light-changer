上のボタンからリポジトリを追加してください。<br />
VCC / ALCOM から対象のプロジェクトの**Manage Project** をクリックしてください。<br />
パッケージ一覧から `Poi Light Changer` を見つけ、横にある **+** ボタンを押すとインストールできます。

## アバターへの導入

導入するアバターを右クリックして `Poi Light Changer > Setup`をクリックします。<br />
アバター内にPoi Light Changerが設置され、導入が完了します。

![Setup](/img/setup.png)

:::note
`VRC Avatar Discriptor` が付与されているオブジェクト以外には表示されません。
:::
