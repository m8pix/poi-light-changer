# Install

Poi Light Changer は VPM package として導入する想定です。

## 事前に必要なもの

- Unity 2022.3.22f1
- VRChat SDK - Base
- VRChat SDK - Avatars
- Modular Avatar
- NDMF
- Poiyomi Shader

## 導入手順

1. VCC または ALCOM に VPM repository を追加します。
2. Unity project に Poi Light Changer package を導入します。
3. package 内の `Poi Light Changer.prefab` をアバター配下へ配置するか、`NDMF/Poi Light Changer` から component を追加します。

## VPM repository

- URL: `https://m8pix.github.io/poi-light-changer/index.json`
- GitHub Pages で公開している public repo の root をそのまま使っています。

## 導入後

導入できたら [Getting Started](/getting-started) に進み、対象マテリアルの収集と各セクションの有効化を行ってください。
