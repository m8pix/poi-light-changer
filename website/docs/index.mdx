## プロジェクトの前提

- 対象プラットフォーム: VRChat PC版
- 対象シェーダー: Poiyomi Shader
- 対象ユーザー: Poiyomi を使う VRChat プレイヤー
- build 方式: NDMF / Modular Avatar ベースの非破壊生成
