## セクションごとの補足

- `SSAO` と `Contact Shadow` は variant / version 条件があります
- `Proximity` と `Backlight` は feature mode で material 側の有効化を制御できます
- `Animation = false` の項目は build 時に static 値として material へ直接注入されます
- `Animation = true` の項目は animator / clip の生成対象になります
