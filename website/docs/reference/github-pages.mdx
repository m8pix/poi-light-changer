# GitHub Pages

この public repo は、docs と VPM 配布物を同じ GitHub Pages で公開する前提です。

## 公開されるもの

- docs site
- `index.json`
- `policy.json`
- `*.zip`

## 仕組み

- docs のソースは `website/` にあります
- GitHub Actions が `website/` を build します
- build 出力へ root の `index.json` `policy.json` `*.zip` をコピーしてから Pages に deploy します

## ローカルで確認する

```bash
cd website
npm install
npm run build:pages
npm run serve
```

`build:pages` の出力をそのまま GitHub Pages artifact に使える構成です。
