# License

Poi Light Changer は proprietary license です。

## ライセンス未認証時の扱い

ライセンスが有効ではない場合、build 時に利用できるのは次の機能のみです。

- Lighting
- Light Direction

それ以外の section と、次の機能は build 側で制限されます。

- Parameters
- Presets
- Texture Bake

## 認証の流れ

- Inspector からライセンスコードを登録します
- `policy.json` を使って認証状態を検証します
- offline grace cache を使って一時的なオフライン状態にも対応します

## policy endpoint

- `https://m8pix.github.io/poi-light-changer/policy.json`

認証まわりの状態表示は Inspector の `ライセンス` セクションで確認できます。
