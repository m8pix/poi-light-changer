# はじめに

Poi Light Chnager は非破壊で導入できるライトチェンジャーです。<br />
アバターのライト・マテリアルの設定をゲーム内から操作できます。

## 必要な依存パッケージ

VRChat での使用を想定しています。<br />
導入には以下のパッケージが必要です。

| パッケージ | 推奨バージョン |
|---|---|
| Unity | `2022.3.22f1` |
| VRChat SDK | `最新版` |
| Modular Avatar | `最新版` |
| NDMF | `最新版` |
| Poiyomi Toon / Pro | `9.3〜10.x` |
