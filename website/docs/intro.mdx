# Intro

Poi Light Changer は、Poiyomi Shader を使っている VRChat アバターに対して、
ゲーム内の ExMenu からマテリアル系パラメーターを操作できるようにするギミックです。

build 時に必要な menu / animator / parameter / material clone を生成するため、
アバター本体へ手作業でアニメーターやメニューを組み込む前提ではありません。

## できること

- Lighting の明るさや倍率の制御
- Light Direction の向き切り替えと XY 調整
- Color 系の色味補正
- Proximity の色変化制御
- Backlight の色と強度まわりの制御
- Outline / SSAO / Contact Shadow の条件付き制御

## 目指している方向

- Poiyomi / ThryEditor に近い Inspector 体験
- build 時のみ動く非破壊ギミック
- 配布可能な安定性

## 注意点

- Poiyomi 9.3 未満は動作保証外です。
- Poiyomi の variant や version によって利用できる項目が変わります。
